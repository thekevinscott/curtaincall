# Changelog

All notable changes to this project are documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/). Because curtaincall is pre-1.0 and does not yet strictly follow [SemVer](https://semver.org/), every PR must add an entry under `[Unreleased]` regardless of whether the change touches the public API. See [CLAUDE.md](CLAUDE.md#changelog) for details.

Breaking changes and meaningful deprecations must also add an entry to [MIGRATIONS.md](MIGRATIONS.md).
